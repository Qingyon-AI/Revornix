# 用户主页与公开资料

Revornix 提供公开用户主页，用于展示创作者的公开资料、粉丝关系以及其已发布专栏。该页面同时承担社区发现、外部访问与搜索引擎收录入口的作用。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260406215533026.png)

## 页面地址

公开用户主页的访问地址为 `/user/[id]`。

该页面基于用户 ID 展示公开资料，不依赖专栏发布 UUID。访问者可以从以下入口进入：

- 社区页中的创作者头像或昵称
- 公开专栏页中的作者信息区
- 站外直接访问公开用户地址

## 页面内容

公开用户主页会展示以下信息：

- 用户头像、昵称、身份角色
- 个性签名
- 粉丝数与关注数
- 已发布专栏总数
- 公开资料说明区
- 已发布专栏列表

如果该用户已经发布过专栏，页面下方会继续展示其公开专栏卡片列表，支持继续浏览和翻页。

## 公开专栏列表

用户主页中的公开专栏列表支持：

- 按关键词搜索该用户已发布的专栏
- 分页加载更多专栏
- 直接进入某个公开专栏页
- 直接订阅该专栏

专栏卡片会展示封面、标题、简介、标签、文档数、订阅数，以及专栏播客相关信息。

## 关注关系

公开用户主页提供关注按钮。已登录访问者可以直接在公开用户页完成关注或取消关注，无需先进入私有工作区。

这使得社区浏览、公开专栏阅读、创作者关注形成一条连续链路：

- 在社区页发现专栏
- 进入公开专栏页阅读内容
- 打开作者公开主页
- 继续查看作者其他公开专栏并完成关注

## SEO 与索引

公开用户主页会输出适合公开索引的页面元信息与结构化数据，用于描述：

- 创作者公开资料
- 页面面包屑路径
- 创作者与社区的关系

用户头像、昵称、签名与公开专栏数量会共同构成该页面的主要公开摘要。

<Callout>
公开用户主页仅展示适合公开访问的资料与已发布专栏，不会暴露私有文档、私有专栏或仅限协作成员可见的内容。
</Callout>
