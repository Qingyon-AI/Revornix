# 仪表盘

仪表盘用于集中查看 Revornix 中最常用的个人工作台信息，包括今日总结、热搜、文档状态、标签分布，以及与阅读节奏相关的统计卡片。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260312200944018.png)

## 页面结构

控制面板首页由以下几个区域组成：

- 顶部欢迎区：显示站点标题、简介，以及一条可刷新的随机诗句
- 初始化提示：当默认模型、默认引擎或其他基础设置尚未完成时，会显示初始化配置提示
- 今日摘要区：显示当日文档总结专栏的生成状态和结果入口
- 热搜区：聚合多来源热搜，并提供进入完整热搜页的入口
- 文档状态区：显示未读文档、最近阅读文档、收藏文档三组快捷入口
- 统计图区：显示按日统计的文档收录趋势，以及按标签聚合的文档分布
- 标签区：显示文档标签和专栏标签，点击即可跳转到对应筛选结果

## 初始化提示

控制面板会在页面上方显示初始化配置提示，用于引导新账号完成默认模型、默认文档总结模型、默认引擎等基础设置。

当这些设置尚未完成时，部分依赖默认资源的能力会继续显示提醒，例如今日摘要中的播客或配图相关状态。

## 今日摘要

今日摘要卡片展示当天文档汇总对应的专栏结果。它会直接读取当日汇总专栏，并显示以下信息：

- 专栏标题、描述与封面
- 专栏处理状态，例如等待处理、处理中、处理完成或处理失败
- 播客生成状态
- 快速进入专栏详情页的入口

如果当天尚未生成总结专栏，卡片会显示空状态；如果当日总结依赖的默认播客引擎或默认生图引擎尚未设置，页面会在对应位置显示配置提醒。

## 热搜

热搜卡片会从独立的热搜服务读取当前热点来源，并在控制面板中展示各来源的当前热词条目。页面提供进入完整热搜页的入口，便于继续浏览整站热搜流。

热搜区使用独立服务提供的数据，控制面板承担的是聚合展示入口。

## 文档状态卡片

控制面板包含三张文档状态卡片，分别用于展示：

- 未读文档：显示未读文档总数，并展示最近一组未读文档的堆叠预览；点击后进入未读文档列表
- 最近阅读：显示最近阅读文档总数，并展示最近阅读过的文档预览；点击后进入最近阅读列表
- 收藏文档：显示已收藏文档总数，并展示最近收藏文档预览；点击后进入收藏列表

卡片中的文档预览支持直接进入对应文档详情页。

## 文档统计

控制面板提供两类文档统计视图：

- 按日统计的文档收录趋势图：展示最近一段时间内每天新增的文档数量
- 文档标签分布图：按标签汇总文档数量，并以图表方式展示分布情况

这两组统计可用于快速判断最近的收录活跃度，以及当前知识库内容主要集中在哪些标签主题上。

## 标签入口

页面下方会分别展示文档标签和专栏标签：

- 文档标签点击后会跳转到“我的文档”并自动带上对应标签筛选
- 专栏标签点击后会跳转到“我的专栏”并自动带上对应标签筛选

这部分适合用作高频主题的快捷入口。

## 随机诗句

控制面板顶部会显示一条随机诗句，并支持手动刷新。该卡片用于提供轻量化的信息点缀，不参与文档、专栏或统计计算。

## 数据来源

控制面板当前使用的数据主要来自以下模块：

- 当日文档汇总专栏
- 独立的 [hot-news](../ai/hot-search) 热搜服务
- 文档未读、最近阅读、收藏等个人列表
- 文档日趋势统计与标签统计
- 文档标签与专栏标签集合
