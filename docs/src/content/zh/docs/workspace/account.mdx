# 账户体系

Revornix 原生支持多租户，每个用户都可以拥有自己的账户、资料页和专栏。

Revornix 不只是个人文档库，也支持围绕专栏、订阅、评论和关注关系构建知识社区。

## 用户主页

每个用户都有自己的主页，你可以在这里查看头像、昵称、个性签名、专栏、粉丝和关注信息。

**SEO 页面视角**

你可以关注 TA，也可以在 TA 的专栏中留言。

## 账户设置

<Callout>仅支持三方登录的部署版本会有 微信 / Git 等登录方式。</Callout>

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318185616886.png)

你可以点击界面左下角的用户头像 -> 账户，进入账户设置页面，修改头像、昵称和个性签名。

账户设置页也承载了邮箱修改、密码修改和账户 API Key 管理等账号能力。

## 账户 APIKEY

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260318185651706.png)

你可以在这里创建和管理账户 API Key，并将其用于后续 API 调用。具体接口列表可参考 [API 文档](api)。
