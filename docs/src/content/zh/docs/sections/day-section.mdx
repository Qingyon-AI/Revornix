# 今日专栏

今日专栏用于按日期聚合文档与知识产出，方便持续回顾和追踪每天的内容变化。

<Callout>
注意：今日专栏没有单独的关闭入口，你创建的文档都会自动加入今日专栏。
今日专栏默认开启自动插图，默认按所属时区的每天 `18:00` 调度；如果当天首次创建文档时已经晚于这个时间点，系统会在当天补触发一次。该时间也可以按需修改。
</Callout>

这是一种系统自动维护的专栏类型，不需要手动创建，系统会根据文档收集情况持续更新。

当用户在新的一天创建第一份文档时，系统会自动创建对应的今日专栏，并将当天文档持续归入这个专栏，再进行总结分析。

今日专栏默认同时开启：

- 自动播客
- 自动插图
- 定时触发处理

该定时触发仅在专栏所属日期当天生效，不会在第二天继续重复执行。

这项能力与[通知功能](../workspace/notification)可以联动使用。当前通知任务仅支持事件触发，因此更适合在专栏更新、评论、订阅等事件发生时进行消息提醒。
