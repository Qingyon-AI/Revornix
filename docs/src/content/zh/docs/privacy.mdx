# 隐私协议

Revornix 致力于为用户提供流畅的信息搜集、分析与分享体验。我们非常重视用户隐私与数据安全。由于 Revornix 同时提供开源版本与官方云版本，本隐私协议将分别进行说明。

> 我们承诺 Revornix 将持续保持开源，你可以选择通过**自行部署**的方式免费使用 Revornix 的大部分功能。

## Revornix开源版本

### 数据收集

作为开源软件平台提供方，我们不会主动收集你或你终端用户的个人信息。你部署环境中的数据由你自行控制并自行负责。

同时，如因你的部署、配置、运维或第三方集成导致用户信息泄露，相关责任由实际部署或运营方承担。

### 使用权限

参照项目功能，如果本项目请求以下权限或系统功能，仅用于其对应的功能实现：

- 网络访问：用于加载远程内容或发起 API 请求。
- 本地存储访问：用于保存或读取用户的自定义设置、文档、专栏等数据。
- 通知权限：用于发送提醒消息。我们承诺非必要不进行通知；同时基于 Revornix 的通知解耦设计，用户可以自行决定接收哪些通知，也可以自行配置通知源和通知目标。

### AI 组件数据传输说明

Revornix 的部分功能基于 AI 服务实现。在你使用文档解析、音频转写、内容理解、生成或问答等 AI 能力时，为完成对应功能，相关数据可能会根据你的实际配置被传输至第三方 AI 服务提供商。

可能被传输的数据包括但不限于：

- 你上传的文档内容，包括网站内容、音频转写结果、速记内容、文件中的 Markdown 或其他文本内容
- 你上传或处理的音频文件原始音频内容
- 你在 AI 对话、生成、总结、检索等过程中主动提交的指令与上下文内容

相关数据可能会被发送至你已选择、启用或自行配置的服务提供商，包括但不限于：

- OpenAI
- Google
- 月之暗面
- 字节跳动火山引擎
- 阿里巴巴百炼
- 你自行配置的其他模型或 AI 服务提供商

Revornix 自身不会主动将上述文档、音频或相关内容用于模型训练。若未来我们开始提供自研模型训练或相关能力，我们将及时更新协议并通知用户；在未经你明确授权，或在你明确拒绝的情况下，我们不会将你的相关数据用于训练。

除法律法规另有要求，或为配合有权机关依法提出的要求外，我们不会主动公开披露你的相关隐私内容。你应同时了解并遵守相关第三方 AI 服务提供商各自的隐私政策、数据处理规则及服务条款。

### 第三方服务

本项目依赖一些第三方服务，例如模型提供方、Apple API 等。请参考对应服务的隐私政策。除实现你主动使用的功能所必需的数据处理外，我们不会主动向第三方发送未经用户同意的个人信息。尤其是在使用云端大模型服务时，对话内容、文档内容或音频内容可能会被发送至对应模型提供方处理。用户或经授权的 Revornix 分发商有义务了解并遵守这些第三方隐私政策。

### 开源说明

本项目为开源项目，任何人都可以查看、修改或部署代码。我们鼓励用户自行部署，并根据自身合规要求调整与隐私保护相关的实现。

- 用户 Google 信息：用于 Google 登录等功能。
