# 参与贡献

本项目核心代码基于[项目协议](https://github.com/Qingyon-AI/Revornix/blob/develop/LICENSE)开源，欢迎开发者在遵守协议的前提下参与贡献。

## 如何贡献

### 1. Fork

先 Fork 本项目，再克隆到本地。

```bash
git clone https://github.com/你的用户名/Revornix.git
```

### 2. 创建分支

在本地创建一个分支，用于开发新功能或修复 Bug。

```bash
git checkout -b 新分支名
```

### 3. 开发

在本地完成开发、调试和测试。

### 4. 提交

将修改提交到本地仓库。

```bash
git add .
git commit -m "提交信息"
```

### 5. 推送

将修改推送到远程仓库。

```bash
git push origin 新分支名
```

### 6. 创建Pull Request

在 GitHub 上创建 Pull Request，申请将分支合并到主分支。

### 7. 等待审核

维护者会审核 Pull Request，如需修改，请根据反馈继续调整。

### 8. 合并

审核通过后，Pull Request 会被合并到主分支。

## 注意事项

- 请确保你的代码符合项目的编码规范。
- 提交 Pull Request 之前，请先在本地完成测试，确保代码能够正常运行。
