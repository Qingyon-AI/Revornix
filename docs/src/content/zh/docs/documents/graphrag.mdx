# 知识图谱

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260522170319488.png)

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260522170520258.png)

知识图谱是 Revornix 文档与专栏工作流中的核心结果之一。

文档图谱和专栏图谱的生成方式并不完全相同：

- 文档图谱可以在文档详情页手动提交生成任务，也会显示等待、处理中、失败、成功等状态。
- 专栏图谱会随着专栏处理结果更新，并和专栏文档集合一起变化。
- 全局图谱接口会基于当前用户的文档集合返回图谱结果。

![](https://qingyon-revornix-public.oss-cn-beijing.aliyuncs.com/images/20260522170605512.png)

图谱视图支持以下交互：

- 搜索知识点
- 在「知识节点关联」与「文档节点关联」两种视角间切换：前者展示知识点之间的引用关系，后者把文档作为节点、按共享的知识点连边
- 点击知识节点查看来源文档和对应片段；在文档视角下点击文档节点可直接跳转到该文档
- 在文档图谱和专栏图谱之间查看不同粒度的知识关系
- 通过图例区分节点类型（知识点 / 文档），节点越大表示关联越多

图谱数据来自 Neo4j，中间会结合文档分块和实体关系构建节点与边。
