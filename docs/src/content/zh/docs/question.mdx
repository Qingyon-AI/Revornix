# 常见问题

### 为何web服务的环境变量需要特殊设置？

由于 Next.js 的构建方式，`web` 服务在编译阶段不会直接读取 Docker Compose 里的 `env_file`。

### 产品是免费的吗？

开源版本免费使用，但不包含商业支持，使用时请遵守[开源协议](https://github.com/Qingyon-AI/Revornix/blob/develop/LICENSE)。云版本采用订阅制。

### 如何获取商业支持？

如需商业支持，可通过微信联系：`Kinda0412`。

### 我可以用这个作为我的笔记软件吗？

可以。Revornix 适合作为知识整理和笔记工具使用，但产品仍在持续迭代，建议保持定期备份。若作为核心生产力工具使用，建议先完成自己的稳定性评估。

### 会有桌面版本吗？

产品路线图包含 Web、iOS、微信小程序和微信客服端，桌面端暂未纳入计划。

### 开源版本和云版本有什么区别？

基本完全一致，核心区别在于

- 云版本零配置，注册即用
- 云版本提供商业支持
- 云版本提供变现机制
