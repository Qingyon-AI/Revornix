# Revornix-v0.7.0

<Callout>
考虑到`MinerU`的包结构有着不少的兼容性问题，在这一版本中，本地的MinerU解析网站、文件的方式被删除，后续并非是不再支持，而是将通过独立部署MinerU服务替代，预计会在`v1.0.0`版本正式支持。
</Callout>

## 更新亮点

- 修复通知任务的bug，现在可以正常进行通知
- 轻度重构通知体系，现在通知表单的增改删更加直观容易
- 补充了部分缓存机制，一定程度上减少了token的使用，加快了请求的速度
- 补充了部分关键节点的打印，便于追溯
- 完善部分类型注解
- 优化打标引擎
- 优化前端UI交互
- 修复已知UI样式问题，比如头像大小等
- 优化专栏和文档的工作流
- 项目文件结构组织优化
- 修复登陆页面跳转
- 优化通知模版
- 修复用户时区不同的问题导致的一些bug，比如每日总结、当天专栏时间不匹配等
- 修复用户能力白名单机制
- 删除本地MinerU方式（后续通过独立部署MinerU服务替代）
- 修复部分多语言缺漏
- 优化了部分代码逻辑
- 修复其他已知的bug
