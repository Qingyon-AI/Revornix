# Revornix-v0.6.4

## Highlights

- Optimized the knowledge graph. You can now interact with it through simple clicks, such as viewing the source of a knowledge point and jumping to the corresponding article.
- Added preview support while editing documents in the quick-note version.
- Updated several core library versions.
