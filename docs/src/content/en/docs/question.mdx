# Frequently Asked Questions

### Why do the web service environment variables need special configuration?

Because of how Next.js handles builds, the `web` service cannot read variables passed through Docker Compose `env_file` during build time.

### Is the product free?

The open-source version is free to use, but it does not include commercial support. Please follow the open-source license when using it. The cloud version is subscription-based.

### How can I get commercial support?

For commercial support, contact us on WeChat: `Kinda0412`.

### Can I use this as my note-taking app?

Yes. Revornix can be used as a note-taking and knowledge-management tool, but the product is still evolving, so regular backups are strongly recommended. If you plan to use it as a primary productivity tool, evaluate stability against your own requirements first.

### Will there be a desktop version?

The roadmap includes web, iOS, WeChat Mini Program, and WeChat customer-service interfaces. A desktop version is not planned at this time.

### What’s the difference between the open-source and cloud versions?

They’re nearly identical. The core differences are:

- The cloud version requires zero setup — just register and use
- The cloud version offers commercial support
- The cloud version provides monetization features
